# AISEO Website Style Guide

## Core Visual Idea

The brand communicates:

- strength
- precision
- forward movement
- confidence
- technical competence

The bull symbolizes **power, direction, and dominance in the market**.  
The website should feel like a **modern digital machine**, not a typical agency template.

Overall impression:

Clean  
Sharp  
Technical  
Confident  
Minimal but strong

Think:

**Tech company + financial strength + modern European design.**

---

# Logo Style Interpretation

The logo uses:

- geometric polygon shapes
- deep blue tones
- sharp edges
- forward stance

This gives several strong design cues:

### Geometry
Use angular shapes inspired by the bull.

Examples:

- diagonal section cuts
- polygon masks
- clipped backgrounds
- sharp corner frames

Avoid soft rounded SaaS design.

---

### Direction

The bull faces forward → growth.

Use:

- diagonal layout flow
- left → right motion
- upward visual lines

The page should visually **move forward**.

---

# Color System

Primary color comes from the logo.

### Primary
Deep Bull Blue

Use for:

- buttons
- accents
- frames
- headings

### Background

Prefer:

Light neutral gray  
or  
Soft gradient

Example:

#F5F7FA
#E9EEF3

This keeps focus on the bull blue.

---

### Accent Usage

Use strong blue for:

- CTA buttons
- section dividers
- borders
- hover effects

Avoid too many colors.

The site should feel **disciplined and confident**.

---

# Typography

Font: **Goldman**

Why it fits:

- geometric
- futuristic
- structured
- masculine

Use hierarchy:

### Headings

Goldman Bold

Large  
Wide spacing  
Confident

Example sizes:

Hero: 64–80px
Section title: 36–48px
Subheading: 22–28px

---

### Body Text

Keep simple and readable.

If Goldman feels heavy for long text, pair with:

- Inter
- Manrope
- Source Sans

This improves readability.

---

# Hero Frame Concept

This is a strong idea and can make the site memorable.

### Framed Hero

Create a subtle border around the hero or whole page.

Example concept:

┌─────────────────────────────┐
│                             │
│   HERO CONTENT              │
│                             │
│        BULL VISUAL          │
│                             │
└─────────────────────────────┘

Style of frame:

- thin blue line
- geometric corners
- slightly angled edges

Inspired by:

- trading platforms
- engineering interfaces
- premium tech dashboards

# Bull-Inspired Elements

Use the logo language across the site.

### 1. Section Dividers

Angular shapes:

/––––––––––
-—————––/

Subtle polygon cuts between sections.

---

### 2. Background Patterns

Very subtle.

Low opacity.

Inspired by bull geometry.

Examples:

- polygon mesh
- diagonal lines
- fragmented shapes

Opacity:

2–4%

Barely visible but premium.

---

### 3. Hover Effects

Buttons or cards could react with:

- slight forward movement
- shadow growth
- color intensification

Idea:

The interface should feel **alive and powerful**.

# Section Style

Each section should feel **structured**.

Avoid random layouts.

Preferred pattern:

Title
Short explanation
Key points
Visual proof

Clean spacing.

Large margins.

Premium feel.

---

# Motion Design

Keep animations purposeful.

Good ideas:

- bull reveal on page load
- diagonal wipe transitions
- subtle grid movement
- number counters for results

Avoid:

- flashy effects
- generic template animations

Motion should feel **controlled and strong**.

---

# Imagery Style

Use visuals that match:

- construction
- engineering
- real work
- machinery
- tools
- infrastructure

Avoid:

- stock business people
- smiling office teams
- generic SaaS graphics

Target audience: **Handwerker / builders / real businesses.**

---

Precise
